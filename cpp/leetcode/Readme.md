## Dependencies

This project was built and tested on the Mac OS X platform with the following dependencies:

* CMake = 3.25.1
* Boost = 1.80.0

## Instructions

Do as you would any CMake project to compile:

```
cd cmake
cmake ..
make
```

Run application:

```
cd bin
./cmake_boost_demo.o
```

Run unit test suite:

```
cd cmake/test
ctest
```
